# Cpp-port-scaner

### Introduction
**Cpp-port-scaner** is a powerful tool for scanning ports using cpp language

---

### Usage

#### 1. Installation
```bash
git clone https://github.com/salari-info/Cpp-port-scaner
cd Pscaner
```



### compile in linux:
```bash
g++ -o port-scaner main.cpp
```

### run in linux
```bash
./port-scaner <ip-addres> <start-port> <end-port>
```

or

```bash
./port-scaner
```
